## 1. 결석계 작성 대상

영산고등학교 결석계 작성 기준에 따라 아래 출결 유형만 처리합니다.

| 출결구분 | 처리 여부 | 비고 |
|---------|---------|------|
| 질병결석 | ✅ 처리 | |
| 출석인정결석 | ✅ 처리 | 사유에 **현장체험학습** 포함 시 제외 |
| 출석인정조퇴 | ✅ 처리 | |
| 출석인정지각 | ✅ 처리 | |
| 출석인정결과 | ✅ 처리 | |
| 질병조퇴·지각 등 | ❌ 제외 | 결석계 작성 대상 아님 |

## 2. 나이스 파일 준비

### 2-1. 나이스에서 출결 파일 다운로드

1. 나이스(NEIS) 접속
2. **출결관리 → 월별 출결 현황** 메뉴 이동
3. 조회 후 Excel 다운로드 시 반드시 **「행병합취소」 옵션 선택**

> ⚠️ 행병합취소를 선택하지 않으면 파일을 불러올 수 없습니다.

### 2-2. 결시교시 입력 (출석인정조퇴·지각·결과만 해당)

나이스 Excel 파일에 `결시교시` 컬럼이 있으면 교시를 직접 입력합니다.

- 형식: `조회,1교시,3교시,7교시,종례` (쉼표로 구분)
- 숫자 교시만 인식하며, 가장 빠른 교시~가장 늦은 교시 범위로 자동 설정됩니다.

### 2-3. 증빙서류 입력 (선택)

나이스 Excel 파일에 `증빙서류` 컬럼을 추가하여 아래 값을 입력할 수 있습니다.

| 입력값 | 결석신고서 체크 항목 |
|-------|-----------------|
| `진단서` / `소견서` / `진료 확인서` / `처방전` | ▣ 진단서, 소견서, 진료 확인서, 처방전 등 |
| `공문` | ▣ 관련 공문 |
| `담임교사 확인서` | ▣ 담임교사 확인서 |
| `기타` | ▣ 기타: |
| 빈칸 | 미체크 (학생·학부모가 직접 작성) |

> **결시교시**와 **증빙서류** 모두 나이스 파일에 미리 입력하지 않아도 프로그램 안에서 직접 입력·선택할 수 있습니다.
> - 결시교시: 미리보기 셀 더블클릭 후 `1~6` 형식으로 입력 (하이픈 `-` 아닌 물결표 `~` 사용)
> - 증빙서류: 미리보기 셀 더블클릭 후 드롭다운 목록에서 선택

## 3. 프로그램 실행

`결석신고서_생성기_v2.exe` 파일을 더블클릭하여 실행합니다.

> Windows 보안 경고가 뜨는 경우: **「추가 정보」→「실행」** 클릭

---

## 4. 단계별 사용법

### Step 1 — 학교 정보 입력

화면 상단에서 아래 정보를 입력합니다.

| 항목 | 입력 내용 |
|------|---------|
| 학년 | 드롭다운에서 선택 (1·2·3학년) |
| 반 | 숫자로 직접 입력 |
| 담임교사 | 담임교사 이름 입력 (결석신고서 하단에 자동 기재) |

### Step 2 — 출결 파일 선택

**「파일 선택…」** 버튼을 클릭하여 나이스에서 다운로드한 Excel 파일을 선택합니다.

- 지원 형식: `.xlsx`, `.xls`, `.xlsm`
- 파일 선택 즉시 데이터 미리보기에 자동으로 불러옵니다.

### Step 3 — 데이터 확인 및 수정

미리보기 화면에서 불러온 데이터를 확인합니다.

- 셀을 **더블클릭**하면 내용을 수정할 수 있습니다.
- 증빙서류는 더블클릭 시 **드롭다운 목록**에서 선택합니다.
- 연속된 결석을 하나로 묶으려면 해당 행들을 선택 후 **「선택 병합」** 버튼을 클릭합니다.

자세한 편집 방법은 [5. 데이터 미리보기 편집](#5-데이터-미리보기-편집)을 참고하세요.

### Step 4 — 저장 위치 지정

기본 저장 위치는 **바탕화면 → 결석신고서_출력.hwpx** 입니다.
변경하려면 **「변경…」** 버튼을 클릭하여 원하는 위치와 파일명을 지정합니다.

### Step 5 — 생성하기

**「▶ 생성하기」** 버튼을 클릭합니다.

- 학생 수에 따라 진행 상태가 표시됩니다.
- 완료 후 파일을 바로 열지 여부를 묻는 창이 뜹니다.
- 생성된 HWPX 파일은 학생 1인당 1페이지로 구성됩니다.

## 5. 데이터 미리보기 편집

### 셀 편집

| 동작 | 방법 |
|------|------|
| 편집 시작 | 셀 **더블클릭** |
| 저장 | **Enter** 또는 **Tab** |
| 취소 | **Esc** |

### 결시교시 입력 형식

결시교시 셀은 `시작교시~종료교시` 형식으로 입력합니다.

```
예시: 1~7   (1교시부터 7교시까지)
     3~5   (3교시부터 5교시까지)
     7~7   (7교시만)
```

> `1-6` 처럼 하이픈(-)을 사용하면 인식되지 않습니다. 반드시 물결표(~)를 사용하세요.

### 증빙서류 선택

증빙서류 셀을 더블클릭하면 아래 목록에서 선택할 수 있습니다.

```
(빈칸)          ← 미체크
진단서
소견서
진료 확인서
처방전
공문
담임교사 확인서
기타
```

### 연속 결석 병합

같은 학생의 연속된 결석 행 여러 개를 하나의 기간으로 묶을 수 있습니다.

1. Ctrl+클릭 또는 Shift+클릭으로 **여러 행 선택**
2. **「선택 병합」** 클릭
3. 묶인 항목은 시작일~종료일로 표시됩니다.
4. 취소하려면 해당 행 선택 후 **「병합 해제」** 클릭

> 병합 조건: 같은 학생 · 같은 출결 유형이어야 합니다.

### 컬럼 정렬

각 컬럼 헤더를 클릭하면 오름차순/내림차순으로 정렬됩니다.

## 6. 자주 묻는 질문

**Q. 파일을 불러왔는데 데이터가 아무것도 나오지 않아요.**
A. 나이스에서 다운로드할 때 **「행병합취소」** 옵션을 선택했는지 확인하세요.

**Q. 특정 학생이 목록에 없어요.**
A. 해당 학생의 출결구분이 결석계 작성 대상이 아닐 수 있습니다. [1. 결석계 작성 대상](#1-결석계-작성-대상)을 확인하세요.
현장체험학습 출석인정결석은 자동으로 제외됩니다.

**Q. 결시교시가 비어 있어요.**
A. 나이스 파일의 `결시교시` 컬럼에 값이 없으면 비어 있습니다. 미리보기에서 더블클릭하여 직접 입력하세요.

**Q. 보고일은 어떻게 계산되나요?**
A. 결석 종료일 + 3일로 자동 계산됩니다.

**Q. 생성된 파일이 한글(HWP)에서 열리지 않아요.**
A. 한글 2018 이상 버전에서 HWPX 파일을 지원합니다. 한글 버전을 확인하세요.
