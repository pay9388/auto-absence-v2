# 결석신고서 자동 생성기 v2

나이스(NEIS)에서 다운로드한 **월별 출결 현황** Excel 파일을 입력으로 받아 결석신고서(HWPX)를 자동 생성하는 도구입니다.

## 주요 기능

- 나이스 출결 현황 Excel 파일 직접 입력 (별도 재입력 불필요)
- 결석계 작성 대상 자동 필터링
- 데이터 미리보기 및 인라인 편집 (더블클릭)
- 증빙서류 드롭다운 선택
- 연속 결석 수동 병합 / 해제
- 종류·증빙서류 체크박스 자동 처리
- 담임교사 이름 입력 지원
- 주말 제외 결석일수 자동 계산
- 보고일 = 종료일 + 3일 자동 설정
- CustomTkinter 기반 다크모드 UI

## 결석계 작성 대상

| 출결구분 | 비고 |
|---------|------|
| 질병결석 | |
| 출석인정결석 | 사유에 "현장체험학습" 포함 시 제외 |
| 출석인정조퇴 | |
| 출석인정지각 | |
| 출석인정결과 | |

## 나이스 파일 다운로드 방법

1. 나이스 → 출결 → 월별 출결 현황
2. **행병합취소** 옵션 선택 후 Excel 다운로드
3. 필요 시 `결시교시` 컬럼에 교시 직접 입력 (출석인정조퇴·지각·결과)
4. 필요 시 `증빙서류` 컬럼 추가 후 값 입력

### 데이터 미리보기 셀 편집

| 컬럼 | 입력 방법 | 형식 |
|------|----------|------|
| 결시교시 | 더블클릭 후 직접 입력 | `1~6` (시작~종료 교시) |
| 증빙서류 | 더블클릭 후 드롭다운 선택 | 아래 목록 참고 |
| 그 외 | 더블클릭 후 직접 입력 | 자유 텍스트 |

> Enter / Tab: 저장 · Esc: 취소

### 증빙서류 선택값

| 선택값 | 체크 항목 |
|-------|---------|
| `진단서` / `소견서` / `진료 확인서` / `처방전` | ▣ 진단서, 소견서, 진료 확인서, 처방전 등 |
| `공문` | ▣ 관련 공문 |
| `담임교사 확인서` | ▣ 담임교사 확인서 |
| `기타` | ▣ 기타: |
| 빈칸 | 미체크 (학생이 직접 작성) |

## 사용 방법

1. 학년, 반, 담임교사 이름 입력
2. 나이스 출결 현황 Excel 파일 선택
3. 미리보기에서 데이터 확인
   - 셀 더블클릭으로 수정 가능
   - 증빙서류는 드롭다운 목록에서 선택
   - 여러 행 선택 후 **선택 병합** / **병합 해제** 가능
4. 저장 위치 지정
5. **생성하기** 클릭

## 실행 방법

```bash
pip install -r requirements.txt
python app.py
```

## 빌드 (Windows EXE)

```
build.bat
```

`dist/결석신고서_생성기_v2.exe` 생성됨.
